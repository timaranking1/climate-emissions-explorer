About This Application

Climate Emissions Explorer is a Flask web application that uses open CO₂ emissions data to help users explore and compare country-level emissions.

Data Source

The data comes from the Our World in Data CO₂ and Greenhouse Gas Emissions dataset. The original dataset contains more than 45,000 rows, but this application uses a filtered subset of 5,000 records to meet the assessment requirement of using between 2,000 and 7,000 records.

Database Design

The application stores the data in two linked database tables:

  • Countries: stores country names and ISO codes.
  • Emission Records: stores yearly CO₂, population, and GDP values.

Each country can have many emission records. This creates a one-to-many relationship between the countries table and the emission records table.

Purpose

The purpose of the application is to demonstrate how open data can be loaded into a relational database and displayed through a Flask web application.
